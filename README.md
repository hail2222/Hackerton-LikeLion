# Hackerton-LikeLion

(1. 2. 는 ~/Hackerton-LikeLion 에서)
1. git clone  
 ```git clone https://github.com/jmhee28/Hackerton-LikeLion.git```

2. 가상환경 키기  
 ```source myvenv/Scripts/activate```

3. 장고 키기  

 ```cd GUAKKIRI```  

 ```python manage.py runserver```
